# Plan Files

Quy ước từ bây giờ:
- Mỗi khi user gửi lệnh `plan`, tạo 1 file mới trong thư mục `plan/`.
- Tên file theo format: `YYYY-MM-DD-HHMM-<slug-ngan>.md`.
- Nội dung file gồm: mục tiêu, phạm vi, các bước triển khai, rủi ro, tiêu chí hoàn tất.

Ví dụ:
- `2026-03-09-0945-phase-filter.md`
- `2026-03-09-1010-priority-edit-popup.md`
