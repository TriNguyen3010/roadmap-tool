# Setup Plan Workflow

## Mục tiêu
- Tạo thư mục `plan` để lưu mọi bản kế hoạch.
- Chuẩn hóa cách lưu file plan cho các lần yêu cầu tiếp theo.

## Thực hiện
1. Tạo thư mục `plan/`.
2. Tạo `plan/README.md` mô tả quy ước đặt tên và nội dung.
3. Cam kết: mỗi lần user gửi lệnh `plan`, tạo 1 file mới theo quy ước.

## Tiêu chí hoàn tất
- Có thư mục `plan/`.
- Có file hướng dẫn trong `plan/README.md`.
- Có file setup ban đầu cho workflow này.
